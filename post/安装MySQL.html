安装MySQL

下载安装包：
下载网址：
https://dev.mysql.com/downloads/

安装

添加环境变量

  • 我的电脑->右键->属性->高级系统设置->环境变量

  • 选择Path->编辑->新建，添加：你的mysql安装文件下的bin文件夹地址

  • 在安装目录下新建my.ini文件（文本文档）

  • 编辑my.ini文件，注意替换你自己电脑上的路径

[mysqld]
basedir = D:\environment\mysql-5.7.31\  //把这两行路径改成自己的安装路径
datadir = D:\environment\mysql-5.7.31\data\
port = 3306
skip-grant-tables

  • 启动cmd(用管理员身份)

    mysqld -install
mysqld --initialize-insecure --user=mysql

    这步完成后，在文件夹里会生成一个data文件（如果没有就自己加一个）

完成后会显示MySQL 服务已经启动成功

然后通过命令mysql -u root -p，进入mysql管理界面（密码为空）

ALTER USER 'root'@'localhost' IDENTIFIED BY '111111';

输入exit 退出
